Install Alexandria

Step 0: Prerequisite - Download Alexandria Plugin

Visit our Downloads Page to find the latest version of Alexandria as well as Grasshopper files for both Builder and Pattern Libraries

Step 1: Open Rhino

Open a new or existing Rhino file

Step 2: Launch Grasshopper

In your Rhino command bar, type “Grasshopper” and open a new or existing Grasshopper file

Step 3: Open User Objects

In Grasshopper, navigate to File > Special Folders > User Objects Folders, open it

Step 4: Extract Alexandria Files

In a new Finder (Mac) or File Explorer (Windows) tab, extract the .zip file for Alexandria

Step 5: Add Alexandria to Grasshopper

Copy and paste the contents of the extracted Alexandria Folder into your Grasshopper User Objects Folder

Step 6: Alexandria is Ready to Go

You have now installed Alexandria on your computer

To begin using Alexandria, navigate to your Grasshopper toolbar and locate the Alexandria tab

For help getting started, visit Getting Started
For addition resources and support, visit our Documentation Hub

*users should repeat this process to install future updates of Alexandria as they are released
