Hexagonal Pattern Array + Attractor

Project Details

This model combines a 2-dimensional pattern component (created using Alexandria) and a scaled curve attractor. The definition is based off the hexagonal grid component, and allows for arrays of any scale. Those geometry arrays then allow a user to apply parametric conditions such as scale, rotation, and more based on a defined curve or point.
